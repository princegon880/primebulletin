
## Features

### User side
- Create account with name, email, and password
- Authenticate user and return a JWT token
- Fetch logged-in user details using token
- View and update their own data

## How To Run ?

For Frontend

```bash
npm i
npm start
```

For Backend

```bash
cd backend 
npm i
nodemon ./index.js
```
